📊 Multidimensional Data Visualization Dashboard

Interaktywny system do analizy i wizualizacji wielowymiarowych danych z wykorzystaniem wykresów równoległych. Aplikacja umożliwia eksplorację danych poprzez dynamiczne filtrowanie i zaawansowane techniki wizualizacji.

Kluczowe funkcjonalności

📈 Wizualizacje

  • Parallel Coordinates
  • Parallel Categories
  • Tabela danych
  • Interaktywne filtry
